# FlexiTrip Admin Panel

A React-based admin panel for FlexiTrip management system.

## Features

- Dashboard with statistics
- Trip Management
- Booking Management  
- User Management
- Settings
- Authentication with Redux

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 19
- Redux Toolkit
- React Router DOM
- Bootstrap
- Sass
- Vite
